pub(crate) fn find_projects(root: &Path) -> BTreeMap<String, ProjectInfo>
Expand description

Recursively searches for Rust projects in the specified directory.

This function looks for directories containing Cargo.toml files within the given root directory and returns a map of project names to their respective ProjectInfo.

§Arguments

  • root - The root directory to search for Rust projects.

§Returns

A BTreeMap where the keys are project names and the values are ProjectInfo structs.
